# Content Creation Workflow

**Status**: Placeholder (Full workflow to be created in Step 7)

## Purpose

End-to-end workflow for creating, reviewing, and refining content across multiple platforms with RLHF learning integration.

## Key Features

- Multi-platform content generation (video + text)
- Brand archetype consistency enforcement
- Human-in-the-loop review and approval
- Feedback collection for learning system
- kie.ai API integration for video/image generation
- Parallel content factory capability

## Workflow Plan

1. **Content Brief** - Topic, platforms, goals
2. **Brand Profile Load** - Fetch archetype from Brand Archetype Agent
3. **Content Generation** - Create platform-optimized content
4. **Review & Refine** - Human feedback loop
5. **Final Approval** - User accepts/rejects/refines
6. **Learning Record** - Capture feedback for RLHF
7. **Output Generation** - Export final content

## Integration Points

- References Brand Archetype Agent's brand profiles
- Records feedback to learning data for Analytics Agent
- Optional kie.ai API for AI-generated visuals
- Called by Campaign Orchestrator for batch generation
