# Market Research Workflow

**Status**: Placeholder (Full workflow to be created in Step 7)

## Purpose

Comprehensive market research workflow using Chrome DevTools MCP for trend identification, audience intelligence, and content opportunity analysis.

## Key Features

- Chrome DevTools MCP integration for web research
- Trend lifecycle tracking (emerging → peak → saturation)
- Audience persona development
- Topic deep-dive research
- Content gap identification
- Competitive landscape scanning

## Workflow Plan

1. **Research Brief** - Define research parameters and objectives
2. **Data Collection** - Chrome DevTools MCP research across platforms
3. **Pattern Analysis** - Identify trends, signals, and opportunities
4. **Insight Synthesis** - Translate findings into actionable intelligence
5. **Strategic Recommendations** - Connect insights to content strategy
6. **Persistent Storage** - Save research for cross-agent reference

## Integration Points

- Provides audience intelligence to Content Creation Agent
- Feeds trend data to Campaign Orchestrator for campaign planning
- Informs Competitor Analysis Agent's research focus
- Requires Chrome DevTools MCP configuration
